Cook JavaScript faster with CoffeeScript

Date Fri 08 January 2016 By Naren Arya Tags javascript / js / programming / coding / web / coffeescript / coffeejs

up and running with CoffeeScript

CoffeeScript. It is untouched apple for people who never looked out of JS. CoffeeScript is not a new language. It is a mini-language that compiles a coffeeScript to JavaScript. It is a transpiler. I will justify my title with explaining the few basics of CoffeeScript.

Introduction

In simple words CoffeeScript is a language in which you write constructs those can be straightaway compiled to JavaScript. Then why not write straight in JS. CoffeeScript constructs are similar to Python. Simple and Precise. The straight benefits of using CoffeeScript are:

  • Simplicity.
  • Compatibility with JS libraries.
  • Syntax like Python.
  • Expressive power with Comphrensions.

Install CoffeeScript & reference

visit This is Coffeescript.org site for downloading coffeescript and run it to generate JavaScript.

Example 1

Let us check whether a variable exists and has some value then do something.

Javascript:

if (typeof elvis !== "undefined" && elvis !== null) {
    alert("I knew it!");
  }

CoffeeScript

alert "I knew it!" if elvis?

Example 2

Defining Lambda type functions is pretty fast

Javascript:

var square;
square = function(x) {
    return x * x;
};

CoffeeScript:

square = (x) -> x * x

Example 3

Conditional statements

JavaScript:

var x = 10;
var y = 20;
var z = 30;
var res, res_final;

if(x < y && y < Z)
{
    res = "Y is in between X and Z";
}
else{
    res = null;
}

res_final = res ? res: "Y is not in between";

CoffeeScript:

x = 10
y = 20
z = 30

if x < y and y < z
  res = "Y is in between X and Z"
else
  res = null

res_final = res if res else "Y is not in between"

Example 4

Loops and comphrensions. JavaScipt inherits what all C family lanuages did.

JavaScript

var courses = ['greens', 'caviar', 'truffles', 'roast', 'cake'];
// menu is some function
for (i = k = 0, len1 = courses.length; k < len1; i = ++k) {
    dish = courses[i];
    menu(i + 1, dish);
}

CoffeeScript

courses = ['greens', 'caviar', 'truffles', 'roast', 'cake']
menu i + 1, dish for dish, i in courses

Example 5

Creating objects is with no brace fuss.

JavaScript:

  var kids = {
    brother: {
        name: "Max",
        age: 11
    },
    sister: {
        name: "Ida",
        age: 9
    }
    };

CoffeeScript:

kids =
  brother:
    name: "Max"
    age:  11
  sister:
    name: "Ida"
    age:  9

If you see these differences, you can easily find coding in CoffeeScript is simple as Python. Code is clear and maintainable. No ugly braces. CoffeeScript has a motto saying "CoffeeScript is just JavaScript". So feel free to try it. Code transpiled by CoffeeScript compiles to JavaScript. Use that JS in your projects. Simple isn't it.
